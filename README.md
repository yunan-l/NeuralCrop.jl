NeuralCrop is a differentiable hybrid GGCM that combines the strengths of the state-of-the-art GGCM LPJmL with machine learning approaches. By implementing components in a differentiable form to achieve seamless integration with machine learning methods, NeuralCrop enables end-to-end 'online training', with ML components optimized in tandem with the model dynamics. 

The reaserch paper can be found https://arxiv.org/abs/2512.20177
